YT Music Desktop

A beautiful, native desktop wrapper for YouTube Music.

Download for Windows Download for macOS Download for Linux

Why use YT Music Desktop?

  • Native Feel: Custom window controls and proper OS integration.
  • Media Keys: Control playback seamlessly from your keyboard or system media controls.
  • Lightweight: Built efficiently to keep memory footprint low.
  • Google Sign-in Works: Uses real Chromium so standard Google auth works perfectly.
  • System Shortcuts: Deep-link protocol support (`ytmusic://`).

Not affiliated with YouTube or Google. Made for personal use.
